### (a) Java Example

```java example

import java.util.*;
class Example{
	public static void main(String args[]){
		Scanner sc = new Scanner(System.in);
		int[] a = new int[5];
		for(int i = 0; i < 5; i++){
			a[i] = sc.nextInt();
		}
		System.out.println("Elements in the array are : ");
		for(int i = 0; i < 5; i++){
			System.out.println(a[i]);
		}
	}
}
```
